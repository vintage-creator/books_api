Documentation: Accessing our endpoints

VintageAPI template guide

In your code editor, write the following and insert your ID Token and the requested endpoint.

const userId = "Insert your ID Token";

const url = "Insert Endpoint (URL)";

const options = {

headers: {

Authorization: `Bearer ${userId}`,
  },

};

const app = async () => {

const res = await fetch(url, options);

const data = await res.json();

console.log(data);

};

app();
